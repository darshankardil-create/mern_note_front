# 📝 MERN Notes Application

A **full-stack Notes Management application** built using the **MERN stack**  
(**MongoDB, Express.js, Next.js, Node.js**).

This project focuses on **clean CRUD operations, frontend–backend separation, and a simple, distraction-free UI**, suitable for learning and demonstrating core full-stack fundamentals.

## 📌 Overview

The application is structured with a **clear separation of concerns**:

- **Frontend:** Built with Next.js for routing, rendering, and UI  
- **Backend:** Built with Express.js for RESTful APIs and database operations  

The project demonstrates how a frontend application communicates with a backend API to perform CRUD operations.
